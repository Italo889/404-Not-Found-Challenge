# 404 Not Found Challenge

## Visão Geral
Este projeto é uma versão da página de erro 404, desenvolvida para o desafio do site DevChallenges. O objetivo é criar uma página 404 que seja visualmente atraente e funcional.

## Recursos e Tecnologias
- **HTML:** Estrutura da página.
- **CSS:** Estilização e responsividade.

## Reconhecimentos
Desafio inspirado por [DevChallenges](https://devchallenges.io/).
